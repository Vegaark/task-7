# Image Resizer Tool

## About

This program automatically resizes and converts all images in a folder using Python and Pillow.

## Features

* Automatically creates folders
* Waits until images are added
* Batch resize and convert
* Very simple to run

## Requirements

Install Pillow:

bash
pip install pillow

## How to Use

1. Run the script:

python image_resizer.py


2. A folder named `input_images` will be created.
3. Copy your images into `input_images`.
4. The program will resize them automatically and save them in `output_images`.

## Folder Structure
project/
├── image_resizer.py
├── input_images
└── output_images
